About the Student Database

This application was developed for the Java Bootcamp October 2017. It has the following features:

  1. Many to many relationships between Students and Courses and Insturcotrs
  2. A form to add students to the database
  3. A form to add instrucotrs to the database
  4. A form to add instructors to courses
  5. Thymeleaf fragments, including:
    • A condition to display links only when students have been enterd into the database AND there are still some students to be added to the course
    • A navigation bar that indicates where in the application a user is
